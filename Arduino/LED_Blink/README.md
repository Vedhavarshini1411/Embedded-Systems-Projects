# LED Blink using Arduino 

## Description

This project demonstrates basic GPIO (General Purpose Input Output) control using an Arduino.

## Tools Used

* Tinkercad
* Arduino Uno

## Components

* LED
* 220Ω Resistor

## Working

The LED connected to digital pin 13 blinks every 1 second using a simple Arduino program.

## Simulation

The circuit was designed and tested using Tinkercad simulation.
